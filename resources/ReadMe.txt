# 概要

　このMODは、AppleMilkTea2とMCEconomy2向けのAddonです。
　AppleMilkTea2とMCEconomy2の両方を導入していないと動作しませんので、ご注意下さい。

　AMT2の方は無しでも動作するような構造に作り替える可能性はあります。
　今はまだアルファ版のため両方が必要です。両方を導入の上でお試し下さい。

　※各アイテムのレシピ確認は、NEIなどのレシピ閲覧機能をご利用下さい。


　このアドオンは主に2つの要素を含みます。
　　・クエスト形式でMCE2のMPを集めていく要素
　　　これは、少額の単発クエストではなく、春夏秋冬の季節単位・あるいは年単位で変化する、
　　　長期的なクエストがメインです。
　　　必要な条件・得られるMP量ともに大味な内容になる予定。

　　・MPを払って機械設備やエネルギーリソースを得る要素
　　　MPの大量消費先兼、さらなるMPを得るための設備投資という感じです。
　　　工場設備については（SS2や各種工業MODなど）他のMODと組み合わせることを想定しています。

　現在アルファ版のため、今ある要素は後者のMP購入機械類のみです。
　また、ショップの価格は暫定版です。ご了承下さい。


# 解説

　●追加されている設備

　　○分電盤
　　　エネルギーシステムのメイン設備です。
　　　「月額チケット」の形で月額料金を払い、ゲーム内30日間AMTチャージが使えるようになります。

　　　使い方：
　　　　・設置する
　　　　　　設置時のプレイヤー向きによって設置方向が変わります。
　　　　　　また、設置すると、このブロックを中心に 3 x 3 の範囲を読み込むチャンクローダー機能があります。
　　　　　　遠出しても安心。

　　　　・月額チケットをチケット用スロットに入れる
　　　　　　左のスロットです。
　　　　　　入れると、使用期限が切られ、利用可能な座標チケット用スロットが解放されます。

　　　　　　期日が来るとチケットは消費されます。
　　　　　　次のチケットは、ホッパーを使って補充が可能。（ほかの補充方法は検討中です）

　　　　　　チケットには、送電量4～16CA/t（CAはChargeAmountの略）、スロット数4or8の種類があります。

　　　　・解放されたスロットに、座標チケットを入れる
　　　　　　座標チケットは、あらかじめAMTチャージの使える装置をスニーク+右クリックで登録しておいて下さい。
　　　　　　チケットを手に持てば、登録座標が視覚的に確認できます。

　　　　　　月額チケットが有効なら、その座標に、月額チケットの送電量に応じてエネルギーが遠隔で飛んでいきます。

　　　　・おまけ
　　　　　　スニーク右クリックすると、盤のふたが開いて中身が見えます。
　　　　　　めちゃくちゃ中身を凝って作ったのに運用時に開ける必要が無いことに気づき、
　　　　　　悲しすぎたので開閉可能にしました。


　　○エネルギータンク
　　　エネルギーをいくらか貯めておけるタンクです。

　　　　　面特性があり、

　　　　　側面の操作盤っぽい面：送電面
　　　　　それ以外の面：受電面

　　　　　となっています。
　　　　　受電面からは、隣接のチャージ装置に16CA/tでエネルギーを送ります。

　　　　　また、他MODエネルギーうちEUを直接扱えます。
　　　　　（デフォルトで出力32EU/tなので、バットボックス程度の機能です。）


　　○エネルギーモーター
　　　AMTチャージ/EUを受け取って、RF/GFに変換できる装置です。

　　　出力は、バッファがある時はそれを消費して32CA/t、無いときは流入チャージ量が出力になります。
　　　よって出力が少し変動するのでお気を付け下さい。

　　　対他MODのエネルギー換算は、
　　　　RF…32CA→320RF
　　　　GF…32CA→96GF・石ギア相当です。


　　○ディーゼル発電機
　　　色々な油を燃やして発電する機械です。

　　　　・使える油
　　　　　今はAMTの油だけです。そのうち増やします。

　　　　　　椿油：8CA/t
　　　　　　植物油：4CA/t

　　　　油は自家製でも良いと思いますし、ショップでも200mB入りの容器を売っています。


　　○アルファ版ショップ
　　　見るからに仮のショップです。
　　　暫定価格の追加アイテムを売っています。どれも高価ですが、
　　　リリース版でもだいたいこれくらいになると思います。

　　　（不定形）作業台＋AMTの茶の葉
