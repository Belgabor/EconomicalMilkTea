#### EconomicalMilkTea! version 1.2a for Minecraft 1.7.10 ####


# 前提Ver&Mod

・Minecraft1.7.10 (Client)
・MincraftForge 1.7.10 10.13.2.1448 以上
　※Forge1448以上でも、ForgeのVerによっては不具合を生じることがあります。
・MCEconomy2 2.5.1以上

　MCEconomy2向けのAddonです。
　AppleMilkTea2っぽい名前ですが、前提にAMT2は不要です。
（もちろんAMT2との連携機能は多くありますが、バニラ+MCE2でも機能するように作ってあります）

　※各アイテムのレシピ確認は、NEIなどのレシピ閲覧機能をご利用下さい。


# 使用方法

・MincraftForgeの導入後に作成されるmodsフォルダの中に、
　MCEconomy2とともに、DLしたファイルを.jarファイルのまま置いてください。

# 概要

　プレイヤーの販売活動を助ける色々な設備をMPショップで展開する『EMT商事』から設備を買い、
　MP経済生活を営んでみよう！というアドオンです。


　○こんなことができる

　主に作物や植物の納品を要求するオーダーシステムにより、
　農工生活に計画性を与えたり、MPを使って施設を拡張できるようにしよう、というアドオンです。

　MPを使って施設拡張やエネルギー源確保が出来るようにしたり、道具や食べ物を買えるようにすることで、
　手作業主体・工業主体どちらにも対応可能に…と考えています。

　また、オーダーの納品自体は、所定の装置に必要数の納品アイテムを入れておけば期日になると自動で精算され、
　煩雑な操作を要求しないので、他のMODプレイと十分平行して遊べると思います。

　プレイスタイルによって"どのタイプのオーダーに力を入れ、どこに拠点を据えるか"も変わってくると思いますので、
　お好みの遊び方でお楽しみ下さい。


　○このアドオンは主に以下の要素で構成されています。。

　　1. 表示されるオーダーに沿ったアイテムを納品することで、MCE2のMPを集めていく要素
　　　ゲーム内1日の少額オーダーから、120日ごとに精算される長期オーダーまであります。
　　　主に作物や植物を要求するものが多く、農耕生活支援の側面が強めです。

　　2. MPを払って機械設備やエネルギーリソースを得る要素
　　　MPの大量消費先兼、さらなるMPを得るための設備投資という感じです。
　　　工場設備については（SS2や各種工業MODなど）他のMODと組み合わせることを想定しています。

　　3. 得たMPの消費先としてのショップや販売物
　　　バニラ産を中心とした食べ物、AMT柚子エネルギー装置が中心です。
　　　また、オーダーをクリアしていくうちに、建築補助アイテムや他MODツールの一部なども購入できるようになります。

　　　特殊効果アイテムなどいくつか。このアドオンでは得られる金額も大きい分、
　　　販売物の価格も高額に設定しています。


　○簡易チュートリアル

　　1. 販売設備ショップをクラフトで作成

　　　　このAddonの最初の基本設備です。序盤に必要な設備を販売しています。
　　　　作業台とリンゴ4個でクラフト。

　　2. オーダーをこなそう！

　　　　まずは販売設備ショップで「取引ボックス」を購入してみましょう。
　　　　しばらく待つと、GUIに現在の取引オーダーが表示されます。
　　　　この「オーダー」をこなして、MPやおまけアイテム（スタンプ）を集めます。

　　3. 集めたMPで買い物

　　　　販売設備ショップでは、MPを使って買い物をするためのショップブロックを5種類売っています。
　　　　柚子エネルギー設備や食べ物、ショップ運営用の小物などを購入できます。

　　4. スタンプを景品と交換

　　　　「オーダー」をGood判定でクリアするとスタンプをもらえます。
　　　　スタンプギフトカタログは、GUIを開いてスタンプを投入することでポイントに応じた景品と交換できます。
　　　　ここでしか入手できない2種類のショップもあります。


# 解説

　●オーダー・MP関連設備

　　○オーダーの仕組み

　　　オーダーは、一日に一度だけある納品判定のタイミングで精算される、アイテム取引システムです。
　　　プレイヤーはこのシステムを使ってEMT商事に遠隔でアイテムを買い取ってもらうことで、多くのMPを入手できます。


　　　・オーダーの種類

　　　　オーダーは4種で、日替わり(Single)、週間(Short)、季節(Middle)、年間(Long)です。
　　　　それぞれ、締め切りが1日、7日、30日、120日です。

　　　　短いものは、クラフトされたアイテムや、中間素材などを要求する注文が多いです。
　　　　長いものは、農作物を中心に要求する注文がメインです。

　　　　オーダーは、ワールド内の日付、チャンク座標、バイオームから決定されるため、
　　　　同一チャンクであれば同じオーダーが表示されます。


　　　・バイオームによる違い

　　　　作物が育ちにくい環境ほど、高く買い取りしています。
　　　　砂漠は果樹、寒冷地は畑作の作物を買い取るオーダーが多い傾向です。
　　　　また、湿地バイオームは他とは異なり、モブドロップ品を中心に買い取りをしています。


　　　・取引の完了判定

　　　　オーダーは、納品要求量の1/2以上を納品できれば取引成功です。
　　　　1/2未満だと、取引は失敗になり、投入したアイテムが無駄になるのでご注意下さい。

　　　　　1/2個～必要個数未満のとき：Bad判定です。報酬はもらえますが、表示されている額より報酬額が安くなります。

　　　　　必要個数達成～2倍未満：Normal判定です。表示されている報酬額に、実際に投入した個数分の倍率がかかります。

　　　　　2倍以上のとき：Good判定です。表示されている報酬額に、2割のボーナスが加算されます。また、おまけのスタンプがもらえます。
　　　

　　○取引ボックス

　　　GUIを開くと、現在の4種類のオーダーが表示されています。
　　　ここでは、必要なアイテム、報酬額、現在の 納品個数/必要個数 が表示されています。

　　　　・オーダーの仕組み

　　　　　GUI内のインベントリに直接投入するか、隣接するインベントリに入れることで納品できます。

　　　　　朝6時以降に一日一度、オーダーの精算と発行が発生します。
　　　　　オーダーは常に4種で、日替わり(Single)、週間(Short)、季節(Middle)、年間(Long)です。
　　　　　季節は、SS2に準拠し30日ごとに変化します。
　　　　　また、バイオームよっても種類が変わります。基本は、厳しい気候ほど報酬も高額になります。

　　　　　同一チャンク内であれば同じオーダーが発行されているので、
　　　　　長期のオーダーを扱う場合は、取引ボックスの設置チャンクを厳選するのも良いかと思います。


　　　　・MP貯蓄

　　　　　報酬はボックス内に蓄積します。
　　　　　貯蓄上限は9999万9900MPまで。


　　　　・ブロックのオーナー

　　　　　最初に設置したプレイヤーをオーナーとして登録します。
　　　　　以後は、オーナーかサーバーOP以外は操作できなくなります。

　　　　　引き出しボタンを押すと、ボックスに貯まっている報酬を受け取れます。


　　○オーダー掲示板

　　　　・看板型のブロックです。
　　　　　右クリックでGUIを開き、現在地の発行中オーダーを見られます。


　　○ギフトカタログ

　　　右クリックでGUIを開けます。
　　　GUIのスロットにはスタンプのみ投入でき、投入数に応じて報酬アイテムが解放されます。

　　　表示されているポイントを消費して、報酬を入手できます。
　　　ほしい報酬アイテムのアイコンをマウスでクリックして下さい。

　　　一冊に貯めておけるスタンプポイントは9999までです。


　　○手提げ金庫

　　　　・MPを貯められる金庫です。
　　　　　上限は99億9999万9900MPまで。また、MP札専用のインベントリスロットもあります。


　　　　・取引ボックス同様、ブロックのオーナーが設定されます。
　　　　　オーナーかサーバーOP以外はGUIを開けず、破壊も不可。

　　　　　ドロップさせた場合も、オーナー情報や中身が保持されます。


　　　　・GUIの使い方

　　　　　<>や+-ボタンで額を操作し、預け入れ・引き出しボタンでMPを出し入れできます。

　　　　　お札として引き出しするボタンもあります。
　　　　　その場合、入力額を10000MP札・50万MP札束に両替して引き出すことが出来ます。


　　○個人チェスト

　　　　・金庫や取引ブロックと同様のオーナー処理を持ったチェストです。54個まで入ります。（ラージチェスト相当）


　　　　・取引ボックス同様、ブロックのオーナーが設定されます。
　　　　　オーナーかサーバーOP以外はGUIを開けず、破壊も不可。

　　　　　ドロップさせた場合も、オーナー情報や中身が保持されます。


　　○各種ショップ


　　　・販売用設備ショップ

　　　　リンゴと作業台でクラフト出来ます。
　　　　まずはこのショップでオーダー用設備や、他のショップを買ってみるのが第一歩です。


　　　・柚子装置ショップ

　　　　後述の柚子エネルギー設備の取扱いショップです。
　　　　販売品はそこそこに高額なので、MPが溜まってきたあとに、
　　　　他MODの工業設備と組み合わせてさらなる高収入を狙ったり、別MODのエネルギー源にしたい場合に。


　　　・コールドショップ

　　　　お徳用ミルクティーパックの他、生鮮食品を販売しています。


　　　・作物ショップ＆食材ショップ

　　　　バニラ品を中心に食材アイテムを買うことが出来ます。

　　　　また、この2つのブロックは、ホットバーに「社員証」を持っていると什器操作用GUIを開け、
　　　　ディスプレイしたいアイテムを置くことが出来ます。


　　　※以下のショップはスタンプ景品のみ。

　　　・建設チケットショップ

　　　　ブロックの一括設置が出来る建築チケットを販売しているショップです。


　　　・工業用品ショップ

　　　　　他MODのレンチアイテムや一部の中間素材を売っているショップです。
　　　　　取扱いアイテムは今後増やすかもしれませんが、作者が使い道を理解しているものに限ると思います。


　●エネルギー関連設備

　　○分電盤

　　　エネルギーシステムのメイン設備です。
　　　「月額チケット」の形で月額料金を払い、ゲーム内30日間、AMTチャージ(CA)が使えるようになります。

　　　使い方：
　　　　・設置する
　　　　　　設置時のプレイヤー向きによって設置方向が変わります。
　　　　　　また、設置すると、このブロックを中心に 3 x 3 の範囲を読み込むチャンクローダー機能があります。
　　　　　　遠出しても安心。

　　　　・月額チケットをチケット用スロットに入れる
　　　　　　左のスロットです。
　　　　　　入れると、使用期限が切られ、利用可能な座標チケット用スロットが解放されます。

　　　　　　期日が来るとチケットは消費されます。
　　　　　　次のチケットは、ホッパーを使って補充が可能。（ほかの補充方法は検討中です）

　　　　　　チケットには、送電量4～16CA/t（CAはChargeAmountの略）、スロット数4or8の種類があります。

　　　　・解放されたスロットに、座標チケットを入れる
　　　　　　座標チケットは、あらかじめAMTチャージの使える装置をスニーク+右クリックで登録しておいて下さい。
　　　　　　チケットを手に持てば、登録座標が視覚的に確認できます。

　　　　　　月額チケットが有効なら、その座標に、月額チケットの送電量に応じてエネルギーが遠隔で飛んでいきます。

　　　　・おまけ
　　　　　　スニーク右クリックすると、盤のふたが開いて中身が見えます。
　　　　　　めちゃくちゃ中身を凝って作ったのに運用時に開ける必要が無いことに気づき、
　　　　　　悲しすぎたので開閉可能にしました。


　　○エネルギータンク

　　　エネルギーをいくらか貯めておけるタンクです。

　　　　　面特性があり、

　　　　　側面の操作盤っぽい面：送電面
　　　　　それ以外の面：受電面

　　　　　となっています。
　　　　　受電面からは、隣接のチャージ装置に16CA/tでエネルギーを送ります。

　　　　　また、他MODエネルギーのうちEUを直接扱えます。
　　　　　（デフォルトで出力32EU/tなので、バットボックス程度の機能です。）

　　　　　GUIのスロットではアイテムへの充電が出来ます。


　　○エネルギーモーター

　　　AMTチャージ/EUを受け取って、RF/GFに変換できる装置です。

　　　出力は、バッファがある時はそれを消費して32CA/t、無いときは流入チャージ量が出力になります。
　　　よって出力が少し変動するのでお気を付け下さい。

　　　対他MODのエネルギー換算は、
　　　　RF…32CA→320RF
　　　　GF…32CA→96GF・Tierは石ギア相当です。


　　○ディーゼル発電機

　　　色々な油を燃やして発電する機械です。

　　　　・使える油
　　　　　今はAMTの油とウォッカの他、一部工業MODの燃料にも対応しています。

　　　　　　椿油：8CA/t
　　　　　　植物油：4CA/t

　　　　油は自家製でも良いと思いますし、ショップでも200mB入りの容器を売っています。


　●その他の追加要素

　　○建築チケット

　　　右クリック使用でブロックの一括設置が出来る使い捨てのアイテムです。
　　　設置できるブロックは固定で、空中にむかって（ブロックをターゲットせず）右クリックすると、
　　　設置ブロックが切り替わります。

　　　消去(Clear)のチケットはブロックを強制的に消去。

　　　設置する範囲は、チケットを手に持っているときに赤いガイド線が出ます。


　　○村スターターチケット

　　　右クリックで使用すると、その場に小さい家を一件と、村人を2人召喚するチケットです。
　　　村人は農民固定です。

　　　建設チケットと同じく、空中に向かって右クリックすると壁の材質が変わります。
